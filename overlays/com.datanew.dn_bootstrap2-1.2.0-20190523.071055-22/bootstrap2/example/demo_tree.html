属性:

名称类型描述默认值
localdatajson类型获取固定数据。如果与属性url都有值，取localdata。无
urlstring（字符串型）获取数据库数据。如果与属性url都有值，取localdata。无
isOpenboolean（布尔型）加载后是否展开全部节点false
ISLEAFboolean（布尔型）父节点是否不可选择false
rootElementboolean（布尔型）是否添加根节点“全部”false
check:enableboolean（布尔型）设置 zTree 的节点上是否显示 checkbox / radiofalse
check:chkStylestring（字符串型）勾选框类型(checkbox 或 radio）checkbox
check:radioTypestring（字符串型）radio 的分组范围。level
view:dblClickExpandboolean（布尔型）双击节点时，是否自动展开父节点的标识true

事件:

名称参数描述
callback:{beforeClick}treeId, treeNode根据返回值确定节点是否允许单击操作
callback:{onCheck}e, treeId, treeNode捕获 checkbox / radio 被勾选 或 取消勾选的事件回调函数
