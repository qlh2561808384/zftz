名称类型描述默认值
titleString定义弹出框的标题。undefined
contentString定义弹出框主题内容。当url有值时，该值无效。 []
urlString定义弹出框主体内容加载的页面。路径以/开头，为根目录下的全路径。undefined
widthNumber定义弹出框的宽度。'-'
heightNumber定义弹出框主体内容的高度。最大不可超过350px，超出无效。350
showCloseButtonBoolean定义弹出框按钮栏处是否需要关闭按钮。false
fatherBoolean定义弹出框是弹出在当前页面还是父层。false
otherButtonsArray定义弹出框按钮栏处的按钮文字。undefined
otherButtonStylesArray定义弹出框按钮栏处按钮的CSS样式。undefined
bootstrapModalOptionBoolean定义模态框默认参数模态框参数 表格true

模态框参数:

bootstrapModalOption:{}
名称类型描述默认值
backdropBoolean指定一个静态的背景，当用户点击模态框外部时不会关闭模态框。true
keyboardBoolean当按下 escape 键时关闭模态框，设置为 false 时则按键无效。true
showBoolean当初始化时显示模态框。true
remoteBoolean使用 jQuery.load方法，为模态框的主体注入内容。如果添加了一个带有有效 URL的 href，则会加载其中的内容。如下面的实例所示：
<a data-toggle="modal" href="remote.html" data-target="#modal">请点击我</a>		false

事件:

名称参数描述
dialogShownone在调用show方法后触发
dialogShownnone当模态框对用户可见时触发（将等待 CSS 过渡效果完成）
dialogHidenone当调用 hide 实例方法时触发
dialogHiddennone当模态框完全对用户隐藏时触发

方法:

调用语法：modalObject.method(param)
名称参数描述
toggleDialognone手动切换模态框。若模态框为关闭状态则打开模态框，反之若模态框为显示状态，则关闭模态框。
showDialognone手动显示模态框
closeDialognone手动关闭模态框
