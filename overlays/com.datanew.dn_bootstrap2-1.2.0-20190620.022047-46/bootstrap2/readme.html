源码地址：https://172.16.12.211:6666/svn/mavenmodel/front/bootstrap/codes/dn_bootstrap

帐号密码是view/DataNew11!!

应用引入此框架有以下方式：

1，普通项目直接把根目录下面bootstrap2目录直接拷贝到应用中的根目录下

2，maven项目直接在pom.xml文件中直接添加

<dependency>

<groupId>com.datanew</groupId>

<artifactId>dn_bootstrap2</artifactId>

<version>1.2.0-SNAPSHOT</version>

<type>war</type>

</dependency>
