minView
Number, String
日期时间选择器所能够提供的最精确的时间选择视图,
0 or 'hour',时分视图
1 or "day",日期视图
2 or 'month'，月份视图
3 or "year",年份视图
4 or "decade",
0
format
String
日期格式， p, P, h, hh, i, ii, s, ss, d, dd, m, mm, M, MM, yy, yyyy 的任意组合,
注:应优先设置minView,否则格式为format : 'hh-ii-ss',minView:2,则选取不到具体时间,最多 选择至日期
"yyyy-mm-dd"
range
Boolean/String
开启左右面板范围选择
如果设置 true，将默认采用 “ - ” 分割。 你也可以直接设置 分割字符。
false
min/max
string
最小/大范围内的日期时间值
设定有限范围内的日期或时间值，不在范围内的将不可选中。
min: '1900-1-1'、max: '2099-12-31'

Boolean

boolean
是否显示底部栏

工具按钮

['clear', 'now', 'confirm']

onLoaded
Function
控件初始打开的回调，返回参数：初始的日期时间对象
onChange
Function
日期时间被切换后的回调。回调返回四个参数（value, oldValue, date, endDate），分别代表：生成的值、旧值、日期时间对象、结束的日期时间对象（range: true才会返回）
onDone
Function
控件选择完毕后的回调。点击日期、清空、现在、确定均会触发。回调返回三个参数（value, date, endDate），分别代表：生成的值、日期时间对象、结束的日期时间对象（range: true才会返回）
