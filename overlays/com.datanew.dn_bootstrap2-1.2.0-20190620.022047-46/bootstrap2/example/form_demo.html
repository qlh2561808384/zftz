名称
参数
描述
submit
{}
ajax提交表单 参数为ajax 参数 url 为必填 默认 type : "POST",dataType : "json" 除data参数 其余ajax参数可进行重写
upload
function
上传文件方法 如使用该方法 需在form标签内 写上action 和method 参数为回调方法 请求效果点击上传按钮后查看请求数据
clear
none
清空表单内所有表单值
disabled
none
禁用表单内所有表单元素【弃用】
enable
none
启用表单内所有表单元素【弃用】
setDisabled
true / false
禁用/启用 表单内所有表单元素（禁用后使用getData无法获取表单数据）
setReadonly
true / false
只读/可用 表单内所有表单元素
load
object
根据表单name 加载各表单数据 主意：如果需要实现 初始化form后 立即调用load 方法 请将该代码移入
validate
none
表单验证
getValueByName
string
根据某个组件的name获取组件值
setValueByName
string
根据某个组件的name设置组件的值
executeByName
string
根据某个组件的name执行某项操作，function(name, method, value){}，参数为name值、方法名、方法对应的参数(没有参数则省略)
