属性:

名称类型描述默认值
urlstringajax请求地址无
datajsonajax请求参数无
asyncboolean默认设置下，所有请求均为异步请求。如果需要发送同步请求，请将此选项设置为 false。默认值: true
typeString请求方式 ("POST" 或 "GET")， 默认为 "GET"。注意：其它 HTTP 请求方法，如 PUT 和 DELETE 也可以使用，但仅部分浏览器支持。默认值: "post")
showLoadingboolean是否显示加载中图标默认值: true)
cacheboolean是否启用缓存默认值: true，dataType 为 script 和 jsonp 时默认为 false。
timeoutNumber设置请求超时时间（毫秒）。此设置将覆盖全局设置。无
beforeSendfunction(xhr)发送请求前可修改 XMLHttpRequest 对象的函数，如添加自定义 HTTP 头。无
successfunction(data)ajax请求后成功方法无
completefunction(XHR, TS)请求完成后回调函数 (请求成功或失败之后均调用)。参数： XMLHttpRequest 对象和一个描述请求类型的字符串。无
redirectboolean表单提交后跳转到新页面false
